Функциональные тербования.
 
TetragonChecker. Модуль для определения типа четырехугольника.
Входные данные (3 параметра): пары значений длин противоположных сторон (считаем что противоположные стороны всегда равны) и угла между двумя соседними сторонами.

Функциональность:
	3.1. Определять: Прямоугольный, Вырожденный, Неизвестный, Квадрат, Ромб, Параллелограмм.
	3.2. Выбрасывать исключение, если длина хотя бы одной из сторон меньше нуля.
	3.3. Выбрасывать исключение, если угол между двумя соседними сторонами меньше нуля или больше 360.
 
 
Параметры:
	A - угол между двумя соседними сторонами.
	X и Y - длины сторон параллелограмма.
 
 
Клпссы эквивалентности:
	1. Некорректные данные - (X < 0) || (Y < 0) || (A == 180) || (A > 360) || (A < 0).

	2. Квадрат - (!Некорректные данные) && (A == 90) && (X == Y).

	3. Прямоугольник - (!Некорректные данные) && (A == 90) && (X != Y).

	4. Ромб - (!Некорректные данные) && (A != 90) && (X == Y).

	5. Параллелограмм - (!Некорректные данные) && (A != 90) && (X != Y).
 
Значения Параметра(ов) -> Ожидаемый Результат:
 
	1. 1, 1, 90 -> квадрат.
	2. 1, 1, 100 -> ромб.
	3. 1, 2, 90 -> прямоугольник.
	4. 1, 2, 100 -> параллелограмм.
	5. 1, -2, 90 -> некорректные данные.
	6. -1, 2, 90 -> некорректные данные.
	7. 1, 2, -90 -> некорректные данные.
	8. 1, 2, 180 -> некорректные данные.
	9. 1, 2, 365 -> некорректные данные.
