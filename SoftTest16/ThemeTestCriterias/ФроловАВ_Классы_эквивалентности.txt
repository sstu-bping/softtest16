Функциональные тербования.

TetragonChecker. Модуль для определения типа четырехугольника.
Входные данные (3 параметра): пары значений длин противоположных сторон (считаем что противоположные стороны всегда равны) и угла между двумя соседними сторонами.
Функциональность:
3.1. Определять: Прямоугольный, Вырожденный, Неизвестный, Квадрат, Ромб, Параллелограмм.
3.2. Выбрасывать исключение, если длина хотя бы одной из сторон меньше нуля.
3.3. Выбрасывать исключение, если угол между двумя соседними сторонами меньше нуля или больше 360.

Прим. авт.: не поленился загуглить признаки параллелограмма:
Если в четырехугольнике противоположные стороны попарно равны, то этот четырехугольник — параллелограмм.
Также необходимо выбрасывать если угол между двумя соседними сторонами равен 180, т. к. таких четырехугольников не существует.

Параметры:
X и Y - длины сторон параллелограмма.
A - угол между двумя соседними сторонами.

Клпссы эквивалентности:
1. Некорректные данные - (X < 0) || (Y < 0) || (A < 0) || (A == 180) || (A > 360).
2. Квадрат - (!Некорректные данные) && (X == Y) && (A == 90).
3. Ромб - (!Некорректные данные) && (A != 90) && (X == Y).
4. Прямоугольник - (!Некорректные данные) && (X != Y) && (A == 90).
5. Обычный параллелограмм - (!Некорректные данные) && (X != Y) && (A != 90).

Значения Параметра(ов) -> Ожидаемый Результат:

1. 1, -2, 90 - некорректные данные.
2. -1, 2, 90 - некорректные данные.
3. 1, 2, -90 - некорректные данные.
4. 1, 2, 180 - некорректные данные.
5. 1, 2, 365 - некорректные данные.
6. 1, 1, 90 - квадрат.
7. 1, 1, 100 - ромб.
8. 1, 2, 90 - прямоугольник.
9. 1, 2, 100 - обычный параллелограмм.
