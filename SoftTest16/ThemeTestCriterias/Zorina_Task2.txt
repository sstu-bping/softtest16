Модуль: Модуль для сложения чисел.
Входные данные (1 параметр): Строка с вещественными числами и с разделителями.
Функциональность:
1.1. Принимать строку, содержащую от 0 до N вещественных чисел (разделитель дробной части точка "."), разделенных запятой «,» и возвращать в качестве результата их сумму. Например, «1,2» равно «3».
1.2. Строка с числами должна начинаться с числа и заканчиваться числом, т.е. строка не может начинаться с разделителя или заканчиваться разделителем. Например, "1,2,3" это валидная строка, а "1," - невалидная, должно броситься исключение.
1.3. Сложение отрицательных чисел запрещено, бросается исключение.

Классы Эквивалентности:
1) Валидная     - строка с положительными числами и разделителем ",".
2) Невалидная - строка, начинающаяся с разделителя.
3) Невалидная - строка, заканчивающаяся разделителем.
4) Невалидная - строка с отрицательными числами.

Значения параметра(строка) -> Ожидаемый результат:
"1,2,3"	      -> 6 
"20,40,5,20,5" -> 90
",4,54,67"	      -> исключение (начинается с разделителя)
",64,87" 	      -> исключение (начинается с разделителя)
"4,75.3,764,"   -> исключение (заканчивается разделителем)
"67,34,2,"	      -> исключение (заканчивается разделителем)
"-45,64,-567"  -> исключение (содержит отрицательные числа)
"657,-8,-12"    -> исключение (содержит отрицательные числа)
